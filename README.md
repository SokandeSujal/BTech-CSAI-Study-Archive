A structured academic repository for B.Tech Computer Science and Artificial Intelligence coursework. The repository is focused on semester-wise academic material such as notes, university-provided PPTs, lab work, assignments, previous year questions, and practice question banks.

The goal is simple: help students find useful study material quickly without searching through scattered folders.

## Public Repository Structure

Only the semester folders and this README are intended to be part of the main student-facing repository.

```text
BTech-CSAI-Academic-Hub/
├── Semester_3/
├── Semester_4/
└── README.md
```

Internal cleanup files, generated reports, and repository metadata are not intended as contribution targets.

## Academic Disclaimer

This is an unofficial student-maintained academic repository.

- University PPTs, PDFs, and official documents belong to their respective university or faculty sources.
- Question banks in this repository are for practice and revision only. They should not be treated as official university question banks unless explicitly marked as official.
- Notes, diagrams, summaries, and handwritten material may include personal interpretation and should be verified against official course material.
- Students should confirm important exam instructions, marking schemes, and syllabus changes through official university communication.

## Contributing

If you find a missing file, incorrect material, broken document, duplicate, or naming issue, please open a GitHub Issue or submit a Pull Request.

Contribution guidelines:

- Add material only inside the relevant semester and subject folder.
- Keep file and folder names clear and without spaces.
- Do not add unrelated repositories or dependency folders.
- Do not mix personal notes with university-provided material.
- Mention the source of the material when possible.
